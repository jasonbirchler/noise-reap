#About

Noise Reap started in 2015 by myself, Cody Iott. It originated from learning how to circuit bend toy keyboards from a co-worker at pizza hut. I got hooked and started learning more and more circuit theory. The first NR module was a utility VCO and sold to someone in Hong Kong.

I quietly burnt out around 2020. I don't really care for experimental music or modular synths anymore. Here's some music that I was a part of, and that I am proud of. Made using homemade modular and noiseboxes, arturia minibrute, and roland alpha juno 2.

I'm currently passionate about programming and web development. I have a fulltime job in a related field. Maybe when I burn out on web dev in 2030 I'll come back and make some purely digital modules.

#Open source policy

Open source repository ↗

Noise Reap modules are open source contingent on three rules:

  1. You can't pretend to be me or use the name Noise Reap. Totally fine with you using the existing module names though.
  2. If you plan to make money off this work you must give 6% profit* to a charity/cause of your choice. Might I suggest something having to do with climate change or systemic rascism.
  3. Don't ask me anything. It's up to you to figure everything out.

*Defined strictly as: profit = customer order total - shipping - parts cost

##Mono Obscura

Modulator-less monosynth

Description

Mono Obscura is a modulator-less monosynth. It combines additive and subtractive synthesis techniques into something fresh and complex. It excels at bass/lead lines but is flexible enough for organic techno sputter and other non-descript allocations.

Firstly, it is a studio ready triangle core VCO. It’s factory tuned to midi note C2 around 65Hz. The TUNE control has a +/-1.25 octave range which also the user to easily shift the root note up or down and octave, but limited enough that it’s not impossible to finely tune.

The VCO also features something called Self Modulation. A function that sends a portion of the triangle output back into its own core. This warps the core to create new waveshapes and asymmetric timbres. It is also destabilizing and will pull the tuning flat at higher settings.

After the VCO comes a ridiculously long 8-stage wavefolder. This folds the triangle in on itself to create new upper harmonics and sizzle.

After the 8-stage folder comes another wavefolder (MORE knob). This one is fundamentally different in nature and combines a new wavefolding circuit with high gain to create something that simultaneously overdrives and wavefolds.

The wavefolders sacrifice the fundamental in exchange for upper timbre complexity. Use BLEND to control how much bass and power you want to bring back to the signal path. Maybe not an exciting feature, but musically it unlocks a LOT of useability.

Next in the signal path is the understated TONE control. This is actually a 24db/oct lowpass filter. There’s some default resonance that give vibrance a hint of “wob” but without being cheesy. Approach this like you would a LPG. It tames and smooths the sound in an organic way. You can manually control the resonance via CV input from none to self oscillation.

Lastly in the monosynth architecture comes a VCA. This is a nice discrete transistor VCA with a linear response.

Specs

  • 16hp
  • 55mA @ +12V
  • 60mA @ -12V
  • 22mm depth

##Paradox

Dual intermodulating VCO

Description

Paradox is a pair of triangle core VCOs.

They can be operated independently. Or, through a diverse set of modulation capabilities, work together to create a wealth of analog FM sounds.

Interesting details

Self modulation

Self Modulation is a technique borrowed from the Bermuda VCO. It sends a portion of the top oscillator’s output back into its own core. On its own this results in pleasant timbre shaping and light de-tuning. When used with other modulation parameters, self mod is an endless source for sweetspots and non-traditional waveshaping.

Super sync

The top VCO can be synced to the bottom VCO or an external source. This is implemented in a haphazard way with no regard for holding waveshape. Think of the the sync as a unique modulation parameter. It creates new waveshapes, gates the core on and off, creates filter sweep effects, and more.

Destructive spirals

The name Paradox is derived from its complex backend of intermodulation and ambiguity. One VCO modulates the other, which modulates the first, which modules the other… These loops can become so intense that the module can be used as a voltage controlled noisebox as well as a traditional VCO.

Specs

  • 8hp
  • 35mA @ +12V
  • 35mA @ -12V
  • 22mm depth

#Book recommendations

If you're interested in starting a small tech craft business in modular synths I recommend the following books in no particular order:

  • The Incredible Secret Money Machine by Don Lancaster
  • CMOS Cookbook by Don Lancaster
  • Electronic Principles by Albert Malvino
  • Transistor Circuit Approximations by Albert Malvino
  • Musical Applications of Microprocessors by Hal Chamberlin
  • AVR Programming: Learning to Write Software for Hardware by Elliot Williams
  • Arduino for Musicians: A Complete Guide to Arduino and Teensy Microcontrollers by Brent Edstrom
